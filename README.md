# betledger

一个纯本地的**体育投注记录与分析**浏览器扩展(Chrome / Edge，Manifest V3）。
框选任意注单截图 → AI 自动识别赔率、本金、结果 → 记入本地账本，提供盈亏、ROI、胜率、日历热力图等分析。

> **性质说明**：本工具是一个**个人记账 / 数据分析工具**，类似把投注记录整理进电子表格。
> 它本身**不提供、不参与、不促成任何博彩或投注活动**，仅帮助用户管理和复盘自己已有的记录。

## 功能

- **AI 截图识别**：框选注单区域，视觉模型自动解析对阵、玩法、赔率、本金、支付额、结果，支持整页多注。
- **多种结算类型**：赢 / 输 / 赢一半 / 输一半 / 走水 / 提前兑现，盈亏一律按「支付额 − 投注额」精确计算。
- **仪表盘**：净盈亏、ROI、胜率、在投敞口、最大回撤、平均 CLV、累计盈亏曲线。
- **注单列表**：搜索、按状态 / 运动 / 平台 / 策略 / 日期筛选，多种排序，分页。
- **分析**：日历热力图(按月切换)、按运动 / 玩法 / 平台 / 策略 / 赔率区间 / 月份分维度盈亏。
- **工具**：赔率转换(欧 / 美 / 港 + 隐含概率)、凯利公式注码、两向套利计算。
- **数据本地**：所有记录仅存于浏览器本地(chrome.storage)，不上传任何服务器；支持 JSON 备份 / 恢复、CSV 导出。

## 解析引擎(任选其一)

在「设置 → 解析引擎」中配置，密钥仅保存在你本地浏览器：

| 引擎 | 说明 |
|---|---|
| 兼容 API（推荐） | 任意 OpenAI 兼容的**视觉模型**，如阿里通义 `qwen-vl` 系列、百度星河 ERNIE-VL 等。填 Base URL + Key + 模型名即可。 |
| 本地 Ollama | 本机运行视觉模型(如 `qwen2.5vl`)，截图不出本机，零 API 费用。 |
| PaddleOCR-VL | OCR 识别 + 内置规则解析，仅需 OCR 令牌。 |

> 视觉模型能理解整页网格布局，识别效果最佳；请务必选择**支持图片输入的 VL / 视觉模型**。

## 安装（开发者模式）

1. 下载或 `git clone` 本仓库
2. 浏览器打开 `chrome://extensions`
3. 打开右上角「开发者模式」
4. 点「加载已解压的扩展程序」，选择本项目文件夹

## 使用

1. 在设置里配置一个解析引擎(填入你自己的 API Key / 令牌)
2. 在注单页面点击扩展图标 →「框选注单区域」
3. 拖拽框选注单，AI 解析后进入「待确认」，核对无误后入库
4. 在仪表盘 / 分析页查看盈亏走势

## 隐私

- 注单数据只存在你本地浏览器，作者与本项目**无法访问**你的任何数据。
